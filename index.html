Typing Test ▶ 3/5
Asteroid destroyer meets typing tutor. This is a fun little game I made to help me practice my typing skills. The game gets progressively harder as you play, and it's a great way to improve your typing speed and accuracy.
Maze Solver ▶ 2/20
Just a little pathfinding visualizer I made to help me understand how different algorithms work. You can choose between breadth first search, depth first search, Dijkstra's algorithm, and A* search. You can also create your own mazes and see how the algorithms perform on them!
Flappy Bird ▶ 2/11
This is a recreation of the popular mobile game "Flappy Bird". The main purpose for the project was learning how to make pixel art using Piskel, adding sound effects from creative-commons licensed music, and saving information in the browser for robust leaderboards.
2 Player Snake ▶ 2/4

Pong ▶ 1/18
This is a recreation of the classic game "Pong". Through some prompt engineering work with ChatGPT, I was able to make a functioning game that has a few extra fun elements added to it.
  • You can choose to play against a computer (very difficult) or a friend!
  • Each time someone gets a point or the ball hits a paddle, the screen shakes a little just to make you FEEL that impact a bit more.
  • The ball increases in speed by 5% everytime it hits a paddle and will increase it's base speed by 10% each time someone wins a point making the game get quite a bit more difficult as it goes on.
